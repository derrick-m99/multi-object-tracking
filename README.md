# Multi-Object Tracking System

Real-time multi-object detection and tracking using YOLOv8 and ByteTrack. Detects and tracks objects across video frames with persistent IDs, bounding boxes, and confidence scores.

## Tech Stack

- Python
- YOLOv8 (Ultralytics)
- ByteTrack (via Supervision)
- OpenCV
